# Support

## Getting Help

Thanks for using the AI Healthcare Platform. Here are the best ways to get help:

### Documentation
- **[README.md](README.md)** — Quick start and overview
- **[docs/](docs/)** — Additional documentation
- **[ARCHITECTURE.md](ARCHITECTURE.md)** — System architecture
- **[SECURITY.md](SECURITY.md)** — Security policies

### Issues
- **Bug reports** — Open an issue using the Bug Report template
- **Feature requests** — Open an issue using the Feature Request template
- **Security vulnerabilities** — See [SECURITY.md](SECURITY.md) for our responsible disclosure process

### Questions
Before asking a question, please check:
1. The existing issues and discussions
2. The documentation in the repository
3. The [CHANGELOG](CHANGELOG.md) for recent changes

### Community
This project is maintained by [rk061215](https://github.com/rk061215). For direct inquiries, please open a GitHub Discussion.

## Response Times

| Inquiry Type | Expected Response |
|--------------|------------------|
| Security vulnerabilities | 24-48 hours |
| Bug reports | 2-3 business days |
| Feature requests | 1 week |
| General questions | 3-5 business days |
